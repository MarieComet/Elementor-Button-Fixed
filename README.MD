Elementor Button Fixed
========================

---------------

* Tags: elementor,widget,button

* Requires PHP: 5.2.4
* Stable tag: 0.0.1

This plugin adds a new widget to the Elementor page builder. This widget display a fixed button.
Once the plugin is installed, you can find the new widget called "Button fixed".
This plugin require the <a href="https://elementor.com/">Elementor page builder</a>.

Description
---------------
This plugin adds a new widget to the Elementor page builder. This widget display a fixed button.
Based on <a href="https://github.com/pojome/elementor-hello-world">this repo</a>.

Installation
---------------
Download the plugin archive and add it to your WordPress installation.
This plugin require the <a href="https://elementor.com/">Elementor page builder</a>.
